# 18.01 Single Variable Calculus

This repository is a dedicated space for documenting my study and implementation of the **MIT 18.01 Single Variable Calculus** curriculum. It combines rigorous mathematical theory with computational exploration.

## 📌 Project Overview

The goal of this project is to archive lecture notes, problem sets, and mathematical proofs while exploring how these concepts can be represented and solved using Python and LaTeX.

## 📂 Key Folders

### `math_in_python/`
This is the core of the repository. It contains:
* **LaTeX Documentation**: Detailed mathematical typesetting for proofs, identities, and calculus problems.
* **Computational Scripts**: Python implementations (using libraries like SymPy) to verify calculations, explore limits, and visualize derivatives/integrals.

## 🛠️ Tech Stack

* **LaTeX**: Used for high-quality mathematical formatting and rendering complex equations.
* **Python**: Leveraged for symbolic mathematics and automated problem-solving.
* **Markdown**: For structured documentation and study logs.

## 📚 Topics Covered

Following the MIT OCW 18.01SC syllabus:
- [x] Limits and Continuity
- [x] Differentiation (Rules, Chain Rule, Implicit Differentiation)
- [x] Applications of Differentiation (Optimization, Related Rates, Newton's Method)
- [x] Integration (Mean Value Theorem, Fundamental Theorem of Calculus)
- [x] Integration Techniques (Substitution, Integration by Parts, Trigonometric Substitution)
- [x] Applications of Integration (Area, Volume, Average Value)
